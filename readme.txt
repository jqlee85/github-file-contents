== Description ==

Syncs contents of files in a GitHub repo to WordPress posts

== Installation ==

= Manual Installation =

1. Upload the entire `/github-file-contents` directory to the `/wp-content/plugins/` directory.
2. Activate Github File Contents through the 'Plugins' menu in WordPress.
